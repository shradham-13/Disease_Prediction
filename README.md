# 🏥 Disease Prediction System

A smart web application where patients can predict diseases based on symptoms and book appointments with the right doctors.

---

## 📌 About The Project

This system connects **Patients**, **Doctors**, and **Admin** on one platform.
- Patient selects symptoms → System predicts disease with **99% accuracy**
- Patient views suitable doctors and sends appointment request
- Doctor accepts or rejects appointments based on availability
- Admin manages and approves both patients and doctors

---

## 👥 User Roles

| Role    | Access |
|---------|--------|
| 🔴 Admin   | Approve doctors & patients, manage entire system |
| 🟢 Doctor  | View appointment requests, accept/reject appointments |
| 🔵 Patient | Select symptoms, get disease prediction, book appointments |

## 🚀 Features

- ✅ 99% accurate disease prediction from symptoms
- ✅ Role-based login (Admin, Doctor, Patient)
- ✅ Doctor listing based on predicted disease
- ✅ Appointment booking system
- ✅ Doctor can accept/reject appointments
- ✅ Admin approval system

## 🛠️ How To Run This Project

### 1. Clone the repository
git clone https://github.com/shradham-13/Disease-Prediction.git

### 2. Go to project folder
cd Disease-Prediction

### 3. Install required packages
pip install -r requirements.txt

### 4. Run the server
python manage.py runserver

### 5. Open in browser
http://127.0.0.1:8000
